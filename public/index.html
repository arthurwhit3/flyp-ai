Sobre o Flyp

F

O Flyp é uma IA desenvolvida pela OrionAI,startup brasileira criada em 2026.

Projeto se deu inicio para fins educacionais e para aprendizagem. Projetada por uma startup solo, o modelo de IA se mantem em constate atualização para testes e projetos futuros. Integrada a LLM, utilizando-se uma API do Google Gemini.

No inicio do desenvolvimento, era apenas possivel rodar no CMD, sem interface visual, o que nao era nada atrativo e pratico.

Hoje, o Flyp é um projeto em desenvolvimento criado para ajudar nas tarefas, dúvidas e estudos do dia a dia. E no futuro pode crescer ainda mais, incluindo histórico de evolução, novas funções, contas de usuário, integração com banco de dados e muito mais.
